📘 Tutorial & Ajuda

Como usar o FPBCIRCUITOS

  1. Arraste uma ou mais Entradas (A, B, C) para o canvas.
  2. Arraste uma Porta Lógica (AND, OR, NOT…) para o canvas.
  3. Arraste uma Saída (LED) para o canvas.
  4. Conecte clicando primeiro num terminal de saída de um nó, depois num terminal de entrada do próximo.
  5. Use os switches no painel direito para definir valores de entrada (0 ou 1).
  6. Clique em ▶ Simular para ver o resultado em tempo real.

Dicas

  • Clique em um nó para selecioná-lo. Pressione Delete para removê-lo.
  • Clique em "Exemplo" para carregar um circuito demonstrativo.
  • A tabela verdade é gerada automaticamente ao simular.
  • NOT aceita apenas 1 entrada; AND, OR, XOR, etc. aceitam 2.
